# Task Manager App

A web-based task management application built with HTML, CSS, Bootstrap, and JavaScript.

## About

This project was developed to practice frontend fundamentals, focusing on DOM manipulation, dynamic rendering, and responsive UI design using Bootstrap.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Responsive layout for mobile and desktop
- Clean and intuitive user interface
